pyliburg
========

Python library to access Hokuyo URG laser range scanners (for Mac and Linux)
Windows is not supported yet.


Install
=======

::

  $ git clone git@github.com:buhii/pyliburg.git
  $ cd pyliburg
  $ git submodule init
  $ git submodule update
  $ python setup.py install
